# 攻略单页 HTML — 结构与风格

## 视觉原则

**目标**：看起来像一个人认真做的旅行手册，不像 AI 批量生成的模板。

### 要

- 干净的排版，充足的留白
- 克制的配色——一个主色 + 一个强调色足够，从行程目的地取灵感（意大利可以是暖赭 + 深蓝，日本可以是墨灰 + 朱红）
- 信息密度高但不拥挤：该折叠的折叠，该省略的省略
- 字体：正文用系统字体栈（`-apple-system, 'Segoe UI', sans-serif`），不额外引入花哨字体

### 不要

- 渐变彩虹背景、霓虹色阴影
- Emoji 堆砌当标题装饰（一两个有意义的 emoji 可以，但不要每行都加）
- AI 套话：「祝你旅途愉快！」「这将是一次难忘的旅程～」「以下是为您精心整理的行程」
- 空洞的 hero 大图 + 半透明遮罩（除非用户提供了真实照片）
- 无意义的动画

---

## 页面骨架

- 单文件：`DOCTYPE`、`<meta viewport>`、`<title>` 含路线关键词
- **页头**：行程名、日期区间、途经城市列表——简短，一目了然
- **按日时间线**：核心内容区
- **页脚**：配图版权 / 免责声明（如有）

---

## 地图规则

### 核心：按城市拆图，跨城不混

每个城市/区域有自己的地图，标注该城市内的活动点位。**不要**把整个行程画在一张大地图上。

### 具体规则

| 场景 | 地图处理 |
|------|---------|
| **同城多日**（如米兰 2 天） | 共享一张城市地图，标注所有点位 |
| **同一天多个活动在同城** | 一张地图，多个标记 |
| **跨城移动（火车/飞机）** | **不画地图**。用文字卡片表达交通方式、时刻、时长 |
| **跨城移动之间的分隔** | 用城市分隔带（城市名 + 简短描述）在视觉上切换语境 |
| **自驾行程的城际路段** | **例外：画完整路线**。自驾时城际路段本身是体验（如多洛米蒂山路、托斯卡纳公路），应展示完整的驾车路线图 |
| **自驾行程的城内活动** | 依然按城市拆图 |

### 地图实现

- **Leaflet**（推荐）：轻量、免费瓦片。每个城市一个 `L.map` 实例，`fitBounds` 到该城市的点位范围
- **Google Maps 嵌入**：用户提供的 `routes[].url` 或 Agent 拼接的 `maps/dir` 链接，`<details>` 折叠 + 懒加载
- **Google 导航按钮**：始终保留，即使嵌入 iframe 受限

### 城市分隔带

在城市切换时插入视觉分隔：

```html
<section class="city-divider">
  <h2>威尼斯</h2>
  <p class="city-subtitle">04.26 — 半日水城</p>
</section>
```

简洁、无装饰——一个城市名 + 日期/一句话足够。

## 单日卡片内容顺序

1. **日期 + 标题**：`Day 3 · 04.26 周日 · 秋叶原 + 台场`
2. **交通**（若有城际移动）：一行文字，不是地图
3. **当日地图**（可选）：该城市内的点位
4. **时间表**：`<ul class="schedule">`
5. **Google 导航按钮**
6. **路线嵌入**（可选折叠）
7. **餐厅预订卡片**
8. **美食推荐**（与餐厅视觉区分：轻量列表或侧栏）
9. **景点折叠**
10. **住宿卡片**（当日入住的酒店）
11. **备注**

## CSS 变量（示例）

```css
:root {
  --bg: #fafaf8;
  --text: #2c2c2c;
  --text-secondary: #6b6b6b;
  --accent: #c45a3c;       /* 从目的地取灵感 */
  --card-bg: #ffffff;
  --border: #e5e5e5;
}
```

不需要每个城市一个主题色。一套配色贯穿全页，干净统一。

---

## 性能与交互

- Google iframe：`<details>` + `toggle` 监听设 `src`，**不首屏加载**
- 图片：`loading="lazy"`；外链先校验 HTTP 200
- 动画：最多用 `opacity` 淡入，不要弹跳、旋转、滑入

## 打印

- `@media print`：隐藏折叠按钮、固定 `day-card` 不透明
- 地图打印为静态截图或隐藏（Leaflet 打印效果差）
